### 🔵 Week 2: Database Management & SQL Fundamentals
*Focus: Structured data manipulation and relational database logic.*

This week was dedicated to understanding how data is stored, queried, and managed using **SQLite** and **DB Browser**. Mastering SQL is critical for analyzing security logs and understanding backend vulnerabilities.

**Key Achievements:**
* **Database Environment:** Created and managed `.db` files using **DB Browser for SQLite**. Built table structures and defined data types.
* **Data Retrieval:** Mastered the core syntax for fetching data: `SELECT` and `FROM`.
* **Advanced Filtering:** Implemented complex queries using `WHERE` clauses combined with logical operators (`AND`, `OR`, `LIKE`). Used `ORDER BY` for sorting and `LIMIT` for data sampling.
* **Relational Logic:** Grasped the concept of **Primary Keys (PK)** and **Foreign Keys (FK)**. Practiced joining multiple tables using `INNER JOIN` and `LEFT JOIN` to consolidate fragmented data.
* **Data Manipulation (DML):** Performed Create, Update, and Delete operations using `INSERT INTO`, `UPDATE`, and `DELETE` commands. 
* **Integrity Management:** Learned the critical importance of the "Write Changes" process to ensure data persistence and prevent corruption.

**🛠️ Week 2 Technical Environment:**
* **Tool:** DB Browser for SQLite
* **Database:** SQLite3
* **Skill:** Database Administration & Querying
